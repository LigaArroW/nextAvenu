
## Старт приложения

1. Перейти в папку server_avenu - (сейчас запускается на 8001 порту, желательно не менять так как api настроена на этот порт) и ввести
```bash
  npm run dev
```
1. Перейти в папку client_avenu - (по стандарту запускается на 3000 порту) и ввести по очередно по мере выполнения запроса

```bash
  npm run build
# далее
 npm run start

```
